Framework

The basic e-mail framework (boilerplate) represents the base code to be used in each e-mail template. It includes the standard HTML code (tested in all major e-mail apps) with commented lines wich need to be pre-filled with relevant data (such lines asta TITLE, INBOX PREVIEW TEXT).

The code provided should be used AS IS and should not be altered no matter what! Any modular data (CONTENT) should be put into the body of the basic framework.
